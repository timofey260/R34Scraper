# Rule34 Smash or Pass

Opens rule34 and gives you a window with 3 buttons: Smash, pass or cursed

* **Smash** saves an image onto some folder and goes to the next image
* **Pass** goes to the next image
* **Cursed** copies image to clipboard

Instructions

```bash
git clone https://github.com/timofey260/R34Scraper
cd R34Scraper
python -m venv .venv
source .venv/activate
python -m pip install -r requirements
python main.py
```
